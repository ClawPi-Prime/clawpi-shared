### Game: Prototype 2 (multiplayer)
Full multiplayer web game built from scratch:
- **Frontend**: HTML5 Canvas, plain JS, Socket.io client, nginx
- **Backend**: Node.js/Express + Socket.io + PostgreSQL
- **k8s manifests** in `/home/clawpi/legion-td-repo/k8s/`

#### Multiplayer features implemented:
- Lobby with 60s countdown, starts immediately on 2 players
- Race selection in lobby (Human Alliance: 6 units)
- Server-synchronized wave start with vote mechanic
- Both players' grids visible simultaneously with interpolated opponent rendering (200ms buffer)
- Game ends when either king dies; disconnect = opponent wins
- Leaderboard via PostgreSQL

#### Human Alliance units (config.js):
- 🗡️ Footman — 15g, tank, armor
- 🏹 Archer — 10g, ranged, fast
- 🐴 Mounted Knight — 30g, charge + knockback every 8s
- ⚕️ Field Medic — 25g, heals most-injured ally in range until full HP
- 💣 Catapult — 60g, splash damage
- ✝️ Priest — 35g, aura (+armor +dmg to allies), follows tankiest unit

#### Mechanics:
